if, elif, else


 - 기본적으로 프로그래밍에서 어떻게 모든게 잘 동작하는지 보여주는 것
 - === : 같음
 - !== : 다름<br>





if 사용법

```
if (condition) {
   block;
} else if {
   block;
} else {
   block;
}
```





and, or


 - and와 or은 피연산자
 - 피연산자란 다른 조건들을 합치게 하는 것&nbsp;&nbsp;<br>
 - &amp;&amp; : and 의미. 조건을 모두 충족할 때<br>
 - || : or 의미. 조건 중 하나를 충족할 때<br>
